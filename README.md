# 끝말잇기 웹게임

봇 또는 친구와 대결하는 한국어 끝말잇기 웹게임입니다.

단어장은 `words/words.txt` 하나만 사용합니다.
단어는 한 줄에 하나씩 넣거나 `/`로 구분해서 넣을 수 있습니다.
플레이어도 이 단어장에 있는 단어만 입력할 수 있습니다.
같은 단어가 여러 번 들어 있어도 게임 로딩 때 처음 나온 1개만 사용합니다.
배포 스크립트를 실행하면 `words/words.txt` 파일에서도 `/` 구분 단어, 중복 단어, 한 글자 단어를 자동으로 정리합니다.

친구 대전은 브라우저끼리 직접 연결하는 PeerJS를 사용합니다.
한 명이 방을 만들고 초대 링크를 보내면, 친구가 링크에서 참가할 수 있습니다.

## 실행

정적 웹사이트라서 GitHub Pages에서 바로 실행할 수 있습니다.
